# PositionManager.jl

## Introduction

## Installation
To install `PositionManager.jl`, press `]` to activate Package mode and type the following command:

```julia
pkg> add PositionManager
```

## Quick start
Assuming `PositionManager.jl` has been installed, start using the package as follows:

```julia
using PositionManager
```

See package documentation for more syntax options.
